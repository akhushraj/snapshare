SnapShare Setup

One-time configuration to connect Google Drive for screenshot sharing.

Step 1 — Google Cloud OAuth required

SnapShare needs an OAuth Client ID to securely access your Google Drive. This is free and takes ~3 minutes.

  1. Go to Google Cloud Console → create or select a project.
  2. Enable the Google Drive API: APIs & Services → Enable APIs → search "Drive API".
  3. Go to APIs & Services → Credentials → Create Credentials → OAuth Client ID.
  4. Application type: Chrome Extension.
  5. Extension ID: loading…
  6. Copy the generated Client ID and paste it below.

⚠️ After saving, you must reload the extension in chrome://extensions for the new Client ID to take effect.

Step 2 — Connect Google Account not connected

Sign in with the Google account whose Drive will store your screenshots. Screenshots go into a folder called SnapShare.

How it works

  • Click the SnapShare icon in your toolbar to capture the full page.
  • Drag to select a sub-region, or annotate the full page.
  • Hit the upload button — the screenshot is saved to your SnapShare Drive folder.
  • A shareable link is copied to your clipboard. Anyone with the link can view it — no login needed.
  • To revoke access, delete the file from your Google Drive.
